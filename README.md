# Express-Boilerplate

Express NodeJs BoilerPlate.

## Installation

For dev -
```bash
npm install
```

For production -
```bash
npm ci --prod
```

## Usage

For dev -
```bash
npm run dev
```
For production -
```bash
node server.js
```

## Usage
```bash
PORT=8000 //default
CLUSTER=FALSE //default
ENV=prod
CUSTOMER_CODE=internal
ENV_CODE=pd
MONGO_HOST=127.0.0.1
MONGO_PORT=27017
MONGO_DBNAME=boilerplate
```

- Mongo
